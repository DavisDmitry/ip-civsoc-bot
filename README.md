# Бот-помощник ФЗИ
Telegram бот-помощник Фракции Защиты Интернета.
## Что умеет:
- Можно связаться с директоратом
## Настройки:
### Переменные окружения:
- `TOKEN` – токен бота от Botfather

Остальные настройки находятся в файле конфигурации `src/config.py`
